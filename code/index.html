WebAssemblyScript (Github)
A simple C-like language which compiles to the WebAssembly binary format. Features strong explicit typing, clean syntax, and near-complete feature parity with WebAssembly, producing binaries up to 90% smaller than the equivalent C++ code. The project includes a token lexer, AST parser, syntax validator, semantic type analyzer, pretty-print formatter, and WASM bytecode generator, all written in TypeScript as a bundled Node module.

Recursive Fibonacci function example: 
export int fibonacci(int n)
	if n <= 1
		return 1
	return fibonacci(n - 1) + fibonacci(n - 2)
